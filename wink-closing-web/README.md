# WINK Bimbel Closing Track Web

Static web version of the WINK/Bimbel closing workflow.

## URL

Open `index.html` directly from the `wink-closing-web` folder, or serve it through GitHub Pages / a static hosting service.

## Included Screens

- Dashboard
- 신청폼
- Master Tracker
- Agent View
- Teacher View
- Manager approval and teacher assignment
- 알림 Queue
- Settings

The current version stores data in browser `localStorage`. For shared team usage, connect the save/load functions to Google Apps Script or another backend.
